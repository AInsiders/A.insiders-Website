Supported Format Conversions

Images

  • WebP → PNG, JPG, GIF, PDF
  • PNG → WebP, JPG, GIF, PDF
  • JPG → WebP, PNG, PDF
  • GIF → WebP, PNG, JPG, PDF
  • BMP → PNG, JPG, WebP, PDF
  • SVG → PNG, JPG, PDF, WebP, GIF
  • TIFF → PNG, JPG, WebP, GIF, PDF
  • ICO → PNG, JPG, WebP, GIF

Documents

  • PDF → Images (PNG, JPG), Text (requires PDF.js)
  • TXT → PDF
  • DOC/DOCX → PDF, TXT (requires mammoth.js)
  • RTF → PDF, TXT, DOCX (requires mammoth.js)
  • ODT → PDF, TXT, DOCX, RTF (requires mammoth.js)

Audio

  • MP3 → WAV (MP3/OGG/AAC/M4A/FLAC/WMA require additional libraries)
  • WAV → WAV (MP3/OGG/AAC/M4A/FLAC/WMA require additional libraries)
  • OGG → WAV (MP3/OGG/AAC/M4A/FLAC/WMA require additional libraries)
  • AAC → WAV (MP3/OGG/AAC/M4A/FLAC/WMA require additional libraries)
  • M4A → WAV (MP3/OGG/AAC/M4A/FLAC/WMA require additional libraries)
  • FLAC → WAV (MP3/OGG/AAC/M4A/FLAC/WMA require additional libraries)
  • WMA → WAV (MP3/OGG/AAC/M4A/FLAC/WMA require additional libraries)

Video

  • MP4 → WebM, AVI, MOV, MKV, FLV + Audio (requires FFmpeg.js)
  • WebM → MP4, AVI, MOV, MKV, FLV + Audio (requires FFmpeg.js)
  • AVI → MP4, WebM, MOV, MKV, FLV + Audio (requires FFmpeg.js)
  • MOV → MP4, WebM, AVI, MKV, FLV + Audio (requires FFmpeg.js)
  • MKV → MP4, WebM, AVI, MOV, FLV + Audio (requires FFmpeg.js)
  • FLV → MP4, WebM, AVI, MOV, MKV + Audio (requires FFmpeg.js)
